# Celebal Summer Internship 2026 -- Week 2 SQL Assignment

## 📌 Project Title

**E-Commerce Sales Database using PostgreSQL**

## 📖 Overview

This project was completed as part of the **Celebal Technologies Summer
Internship 2026 (Week 2)**.

The objective of this assignment is to design and implement an
**E-Commerce Sales Database** in PostgreSQL and write SQL queries to
retrieve, filter, aggregate, and analyze data while understanding
database constraints, relationships, joins, transactions, and ACID
properties.

------------------------------------------------------------------------

## 🛠️ Technologies Used

-   PostgreSQL
-   pgAdmin 4
-   SQL

------------------------------------------------------------------------

## 📂 Database Schema

The database consists of four tables:

-   `customers`
-   `products`
-   `orders`
-   `order_items`

### Relationships

-   One Customer → Many Orders
-   One Order → Many Order Items
-   One Product → Many Order Items

## 🚀 How to Run

1.  Install PostgreSQL and pgAdmin 4.
2.  Create a new database.
3.  Execute the `CREATE TABLE` statements.
4.  Insert the sample data.
5.  Run the SQL queries for Sections A--E.
6.  Verify the outputs.
